# 🚀 TalentHive

Projeto desenvolvido para **treinar responsividade com HTML e CSS**, simulando a landing page de uma plataforma de freelancers chamada **TalentHive**.

O foco principal do projeto é a adaptação entre **Mobile First** e **Desktop**, utilizando boas práticas de organização de layout, variáveis CSS e media queries.

---

## 📱💻 Sobre o Projeto

O **TalentHive** é uma landing page fictícia voltada para:

- Conexão entre freelancers e empresas  
- Divulgação de vagas remotas  
- Plataforma moderna e responsiva  
- Layout limpo e profissional  

O projeto foi construído com:

- HTML5 semântico  
- CSS3 puro  
- Media Queries (`min-width: 1024px`)  
- Mobile First  
- Variáveis CSS (`:root`)  
- Organização modular por seções

## 🖥 Layout Responsivo

O projeto possui duas versões principais:

### 📱 Mobile (padrão)
- Menu hambúrguer  
- Imagens otimizadas para celular  
- Layout em coluna  
- Textos centralizados  

### 💻 Desktop (a partir de 1024px)
- Menu completo visível  
- Layout expandido  
- Imagens diferentes para desktop  
- Ajuste de espaçamentos e tipografia  

Exemplo de media query utilizada:

```css
@media screen and (min-width: 1024px) {
   /* Estilos Desktop */
}
```
